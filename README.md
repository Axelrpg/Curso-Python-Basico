# 🐍 Python Básico - TecNM

Este repositorio contiene mis apuntes, ejercicios y prácticas realizadas durante el curso de **Python Básico** impartido por el TecNM.

El objetivo es reforzar los fundamentos del lenguaje y desarrollar habilidades prácticas en programación con Python.

---

## 📚 Contenido del curso

### 🔹 Tema 1: Introducción a Python y su entorno de desarrollo

* Instalación y configuración
* Uso del intérprete
* Primeros programas

### 🔹 Tema 2: Sintaxis Básica y Manipulación de Datos

* Variables y tipos de datos
* Operadores
* Entrada y salida de datos

### 🔹 Tema 3: Estructura de Control y Flujo de Ejecución

* Condicionales (`if`, `elif`, `else`)
* Ciclos (`for`, `while`)
* Control de flujo

### 🔹 Tema 4: Funciones y Secuencias

* Definición de funciones
* Parámetros y retorno de valores
* Listas, tuplas y diccionarios

### 🔹 Tema 5: Técnicas Avanzadas en la Manipulación de Cadenas

* Métodos de strings
* Formateo de texto
* Procesamiento de cadenas

---

## 🎯 Objetivos

* Comprender los fundamentos de Python
* Desarrollar lógica de programación
* Resolver problemas mediante código
* Preparar bases para temas más avanzados

---

## 🛠️ Tecnologías utilizadas
